# DayOne - Simple PDF Editor

A small browser-based PDF editor built with vanilla HTML/CSS/JS.

## Features

- Load a local PDF file.
- Rotate all pages by 90°.
- Add text on a selected page at custom coordinates.
- Delete a selected page.
- Preview a page after edits.
- Download the edited PDF.

## Run locally

Because this app uses browser scripts and file uploads, run a local server:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.
